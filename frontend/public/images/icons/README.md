# Industry Icons

This directory contains SVG icons for each industry.

## Required Icons

Place the following icon files in this directory:

- healthcare.svg
- finance.svg
- manufacturing.svg
- retail.svg
- education.svg
- logistics.svg
- energy.svg
- telecom.svg
- construction.svg
- automotive.svg
- agriculture.svg
- tourism.svg
- media.svg
- technology.svg
- government.svg
- aerospace.svg
- professional.svg
- food.svg
- chemical.svg

## Icon Requirements

- Format: SVG (preferred) or PNG
- Size: 32x32px or larger (will be scaled to 32x32px)
- Style: Simple, clean, professional
- Color: Can be any color (will be displayed as-is)

## Usage

Icons are referenced in DynamoDB as `/images/icons/{name}.svg` and displayed in the IndustryDetail page.
